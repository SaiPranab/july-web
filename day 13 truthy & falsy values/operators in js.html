Operators in JS

There are mainly 3 types of Operators

  • unary
  • binary
  • ternary

Unary operators

  • Unary ( + )
  • unary ( - )
  • pre & post increment ( ++ )
  • pre & post decrement ( -- )
  • logical NOT ( ! )
  • bitwise NOT ( ~ )
  • typeof
  • delete
  • void

Binary operators

  • Arithmetic operators

      • +, -, *, /, %, **

  • Assignment Operators

      • +=, -=, *=, /=, %=, **=

  • Comparison Operators

      • == (loose equals), === (strict equals), !=, !==, >, >=, <, <=

  • Logical Operators

      • &&, ||

  • Bitwise Operators

      • &, |, ^, <<,>>, >>>

  • Some Special Operators

      • in, instanceof, ?. (optioanl chaining), ?? (nullish coaelscing)

Ternary Operator

  • (1stexpression) ? (2ndexpression) : (3rdexpression)
