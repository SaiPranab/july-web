String in JS

- In JavaScript string represents a sequence of characters.

- We can create the string object in 2 ways

  • By String literal
    • let str = "Hello"
  • By string object (using new keyword)
    • let str1 = new string("Hello")

In JS we can access any single character of a string by doing something like this.

Expression: 'Hello World'[1]

Result: 'e'

In Javascript strings are immutable, which means once a string is created it contends cannot be modified
