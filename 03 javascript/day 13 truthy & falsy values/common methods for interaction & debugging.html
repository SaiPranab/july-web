Common JavaScript Methods for Interaction

  • alert(message) – Displays a popup alert box with a message.
  • prompt(message) – Asks the user for input and returns it as a string.
  • confirm(message) – Shows a dialog box with OK and Cancel buttons, returning true or false.

Common JavaScript Methods for Debugging

  • console.log(message) – Prints general information to the browser’s developer console.
  • console.warn(message) – Prints a warning message to the console.
  • console.error(message) – Prints an error message to the console.
